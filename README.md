# Quick Start


## POSIX

```console
$ ./build.sh
$ ./ded src\main.c
```
**Linux** requires packages install with:
```sudo apt update && sudo apt install gcc libftgl-dev libsdl2-dev libglew-dev```

## Windows MSVC

```console
> .\setup_dependencies.bat
> .\build_msvc.bat
> .\ded.exe src\main.c
```
